# 项目描述

## 项目介绍
- 本项目的目的是从目标文件夹中提取所有 .xlsm 文件中的 Manufacturing Process 条目
- 提取出的条目经过人工审核后存入数据中
- .xlsm 文件的格式参照
- 数据库的具体内容参考 .claude/skills/database 技能

## 开发步骤
1. 构筑项目的 UI
2. 使用实验数据实现业务逻辑
3. 数据库脚本开发
